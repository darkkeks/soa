# Chat client

## Сборка и запуск

```shell
# build server
./gradlew serverJar

# build client for native platform
./gradlew clientJar

# build client for x86 linux
./gradlew -Plwjgl-natives=natives-linux clientJar

# launch server
java -jar build/libs/*-server.jar

# launch client
java -jar build/libs/*-client.jar
```
